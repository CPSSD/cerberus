---

## Requirements:

#### Development
- Rust Nightly
- Protobuf Compiler
- net-tools (if running worker locally)

#### Deployment
- Docker
- Docker Compose

---

## Building the project

Build everything by running:

```
$ cargo build --all
```

---

## System Requirements
The project currently only works on Linux. macOS and other platforms are planned for the future.
